## 二、LSE-v1 模块设计

模块名称：

```python
LineStructureEnhancement
```

简称：

```text
LSE
```

设计目的：

增强 PIDNet-S 中 P branch 的高分辨率细节特征，使模型更容易学习输电线、细杆、横担等细长结构，目标是减少输电线断裂和漏检，同时尽量避免线变粗和大面积假阳性。

LSE-v1 只做轻量结构增强，不改输出、不加新任务、不加新 loss。

---

## 三、LSE-v1 结构

输入输出 shape 必须一致：

```python
input:  [B, C, H, W]
output: [B, C, H, W]
```

结构采用三个 depthwise 分支：

### 1. 水平方向分支

用于增强横向或近似横向的线结构：

```python
Conv2d(C, C, kernel_size=(1, k), padding=(0, k//2), groups=C, bias=False)
BatchNorm2d(C)
ReLU(inplace=True)
```

### 2. 垂直方向分支

用于增强竖向或近似竖向的杆件结构：

```python
Conv2d(C, C, kernel_size=(k, 1), padding=(k//2, 0), groups=C, bias=False)
BatchNorm2d(C)
ReLU(inplace=True)
```

### 3. 局部空洞分支

用于补充局部上下文和斜向结构响应：

```python
Conv2d(C, C, kernel_size=3, padding=dilation, dilation=dilation, groups=C, bias=False)
BatchNorm2d(C)
ReLU(inplace=True)
```

### 4. 融合

```python
cat = torch.cat([h, v, d], dim=1)
fused = Conv2d(3*C, C, kernel_size=1, bias=False)(cat)
fused = BatchNorm2d(C)(fused)
```

### 5. 残差输出

```python
out = x + alpha * fused
out = ReLU(inplace=True)(out)
```

推荐默认参数：

```python
kernel_size = 9
dilation = 2
alpha = nn.Parameter(torch.zeros(1))
```

注意：`alpha` 初始化为 0，使模型初始状态接近 baseline，避免刚开始训练时大幅扰动 PIDNet-S 原有特征。

如果 fusion 的最后 BatchNorm 有 affine 参数，建议将其初始化为 0：

```python
nn.init.zeros_(self.fuse_bn.weight)
nn.init.zeros_(self.fuse_bn.bias)
```

如果项目中的 BN 写法不同，请按实际代码风格处理。核心原则是：LSE 初始行为应接近 identity residual。

## 四、LSE 插入位置

请在 `pidnet_lse_v1.py` 中找到 PIDNet-S 的 P branch 高分辨率细节分支。

通常 forward 中会有类似：

```python
x_ = self.layer3_(...)
x_ = self.layer4_(...)
x_ = self.layer5_(self.relu(x_))
```

请在 `layer5_` 之后、Light-Bag / DFM 融合之前插入：

```python
x_ = self.layer5_(self.relu(x_))
x_ = self.lse(x_)
```

要求：

```text
1. LSE 只作用于 P branch 的 x_。
2. 不作用于 I branch 的 x。
3. 不作用于 D branch 的 x_d。
4. 不替换 PAPPM。
5. 不替换 Light-Bag。
6. 不替换 PagFM。
7. 不改 final_layer。
8. 不改输出类别数。
9. 不改 forward 返回格式。
```
