一、实验背景

当前任务是无人机输电通道语义分割，类别为：

0 background
1 powerline / transmission line
2 wooden tower
3 lattice / steel tower
4 other / tucohy tower
255 ignore

当前 baseline 是 PIDNet-S，使用 Cityscapes 预训练权重，5 类输出，OHEM 训练。之前已经实现并训练过 HDP-DGA-v1，即直接用 DGA-PAPPM 替换原 PAPPM。该版本可以正常训练和推理，但 best mIoU 与 baseline 基本持平，没有形成稳定提升。主要问题包括：

1. DGA-PAPPM 直接替换 PAPPM，导致原 `spp.*` 预训练权重不能继承；
2. 改动位置位于 I branch 深层低分辨率特征，对输电线、细杆塔等细小结构帮助有限；
3. 全局 gate 不能实现空间位置级自适应；
4. 直接替换 PAPPM 牺牲了原 PIDNet-S 稳定的多尺度上下文表达。

因此 v3A 不再替换 PAPPM，而是保留 PAPPM，在 P branch 的 1/8 高分辨率特征上增加一个 zero-init residual 形式的 BMHR 模块。

二、BMHR 模块设计动机

模块名称：

BMHR = Boundary-guided Multi-scale High-frequency Refinement
中文：边界引导多尺度高频细节增强模块

设计依据：

1. powerline 是细长高频结构，容易在下采样中断裂；
2. wooden tower 通常细、小、弱纹理，容易与树影、道路纹理、电线混淆；
3. other/tucohy tower 形态变化大，有时是大横臂，有时是细长单杆，需要多尺度建模；
4. lattice tower 已经较好，不能因为增强其他类别导致 lattice 明显下降；
5. PIDNet 原本有 P/I/D 三分支，其中 P branch 保留细节，D branch 提供边界信息，因此在 P branch 上用 D branch 引导增强比在低分辨率 I branch 里替换 PAPPM 更合理；
6. 高频增强容易误增强树影、草地、道路纹理等背景噪声，因此需要 D branch 做空间 gate 引导；
7. BMHR 必须采用 zero-init residual，保证模型初始行为接近 baseline，降低破坏预训练模型的风险。

五、实现 BMHR 模块

请在 `models/hdp_modules_v3a.py` 中实现：

```python
class BMHR(nn.Module):
    def __init__(
        self,
        channels,
        reduction=4,
        use_boundary_gate=True,
        zero_init=True
    ):
        ...
```

输入：

```python
p: P branch feature, shape [B, C, H, W]
d: D branch feature, shape [B, C, H, W]
```

输出：

```python
enhanced_p: shape [B, C, H, W]
```

对于 PIDNet-S，`channels=128`。

BMHR 内部结构如下：

1. reduce 层

```text
1×1 Conv + BN + ReLU
channels -> mid
mid = max(channels // reduction, 16)
```

2. local branch

```text
depthwise 3×3 Conv + BN + ReLU
```

作用：增强局部细节、细杆、细线。

3. medium branch

```text
depthwise 3×3 Conv, dilation=2, padding=2 + BN + ReLU
```

作用：增强横担、局部塔臂、局部杆塔结构。

4. large branch

```text
depthwise 3×3 Conv, dilation=3, padding=3 + BN + ReLU
```

作用：增强更大范围的杆塔局部上下文和 other 类多形态结构。

5. high-frequency branch

本次不要使用 FFT。
请使用固定 Laplacian 或 Sobel-like high-pass depthwise filter。

要求：

1. high-pass filter 不参与训练，或者初始化为 Laplacian 后 `requires_grad=False`；
2. high-pass 后接 `3×3 depthwise Conv + BN + ReLU`；
3. 该分支用于增强输电线、细杆、杆塔边界和横担边缘；
4. 不要把高频分支做得很重。

推荐 Laplacian kernel：

```python
[[0, -1, 0],
 [-1, 4, -1],
 [0, -1, 0]]
```

也可以使用八邻域版本：

```python
[[-1, -1, -1],
 [-1,  8, -1],
 [-1, -1, -1]]
```

选择一种即可，保持实现简单。

6. spatial gate

如果 `use_boundary_gate=True`：

```python
gate_input = torch.cat([p, d], dim=1)
```

如果 `use_boundary_gate=False` 或 `d is None`：

```python
gate_input = p
```

gate 输出：

```python
[B, 4, H, W]
```

对应四个分支：

1. local
2. medium
3. large
4. high-frequency

使用：

```python
weights = torch.softmax(gate_logits, dim=1)
```

注意：这里必须是空间 gate，即 `[B, 4, H, W]`，不要做成全局 `[B, 4, 1, 1]`。

7. fusion

```python
fused = (
    weights[:, 0:1] * b1 +
    weights[:, 1:2] * b2 +
    weights[:, 2:3] * b3 +
    weights[:, 3:4] * b4
)
```

8. expand 层

```text
1×1 Conv + BN
mid -> channels
```

9. zero initialization

如果 `zero_init=True`，请将 expand 中最后一个 BN 的 weight 初始化为 0。
或者使用 learnable scalar `alpha`，初始化为 0。

目标是使模块初始时近似恒等映射：

```python
BMHR(p, d) ≈ p
```

推荐 forward 逻辑：

```python
res = self.reduce(p)

b1 = self.local(res)
b2 = self.medium(res)
b3 = self.large(res)
b4 = self.high_freq(res)

weights = self.gate(p, d)

fused = (
    weights[:, 0:1] * b1 +
    weights[:, 1:2] * b2 +
    weights[:, 2:3] * b3 +
    weights[:, 3:4] * b4
)

residual = self.expand(fused)

return p + residual
```

在初始化中添加：

```python
if use_bmhr:
    self.bmhr = BMHR(
        channels=planes * 4,
        reduction=bmhr_reduction,
        use_boundary_gate=bmhr_use_boundary_gate,
        zero_init=bmhr_zero_init
    )

    self.bmhr = nn.Identity()
```

对于 PIDNet-S，`planes=32`，因此 `planes * 4 = 128`，与 P branch 最终输出通道一致。

forward 中找到原始逻辑：

```python
x_ = self.layer5_(self.relu(x_))
x_d = self.layer5_d(self.relu(x_d))
x = F.interpolate(
    self.spp(self.layer5(x)),
    size=[height_output, width_output],
    mode='bilinear',
    align_corners=algc
)

x_ = self.final_layer(self.dfm(x_, x, x_d))
```

请改为：

```python
x_ = self.layer5_(self.relu(x_))
x_d = self.layer5_d(self.relu(x_d))

x_ = self.bmhr(x_, x_d)

x = F.interpolate(
    self.spp(self.layer5(x)),
    size=[height_output, width_output],
    mode='bilinear',
    align_corners=algc
)

x_ = self.final_layer(self.dfm(x_, x, x_d))
```
